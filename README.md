# Hoja de Cálculo con Matrices Dispersas basadas an listas simplemente enlazadas

Proyecto Nro. 1 — Algoritmos y Estructuras de Datos  
Implementación de una hoja de cálculo simple usando matrices dispersas con listas enlazadas cruzadas en C++.

---

## Requisitos previos

- [Qt6](https://www.qt.io/download) instalado
- CMake 3.20 o superior


# 1. Clonar el repositorio
git clone https://github.com/Mija9000/MatricesDispersas
cd MatricesDispersas

# 2. Crear carpeta de build
mkdir build && cd build

# 3. Configurar con CMake
cmake .. -DCMAKE_PREFIX_PATH=/usr/lib/x86_64-linux-gnu/cmake

# 4. Compilar
cmake --build .

# 5. Ejecutar
./MatricesDispersas
